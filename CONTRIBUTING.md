# Contributing to ICEP-MVP

ICEP-MVP is reference tooling for the ICEP protocol.

## Development

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .
```

## Quick Checks

- `icep validate examples/intent.json`
- `icep validate examples/event-log.jsonl`

## Scope

Keep changes minimal and focused on validation and event-log tooling.
